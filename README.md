# SkillForge

SkillForge is an Android application built using Kotlin and Jetpack Compose. The app displays course categories, course details, and lesson videos using data fetched from an API.

## Features

- Home Screen with Categories
- Popular Courses List
- Course Detail Screen
- Lesson Player Screen
- Navigation between screens
- API Integration using Retrofit
- Image Loading using Coil
- Loading and Error State
- MVVM Architecture

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM
- Retrofit
- Coroutines
- Coil
- Navigation Compose

## Build Instructions

Clone the repository and open it in Android Studio.

Run the project or build the APK using:

```bash
./gradlew assembleDebug
```

The debug APK will be generated in:

```
app/build/outputs/apk/debug/
```
